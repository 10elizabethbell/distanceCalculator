# distanceCalculator — spin-up

Say **"arthur, spin up my distanceCalculator project"**. macOS menu-bar app
(Swift + C launcher) that shows traffic-aware drive time to any city via
MapKit. One Ghostty window per plain line (opened at this directory);
`@run` = headless; `@open` = macOS open.

```spinup
claude
# launch the built menu-bar app (🚗 icon)
@run open DistanceCalculator.app
```
